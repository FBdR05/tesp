Copy these three updates to patch PYPOWER for deprecated behaviors, 
primarily the use of floats for array indices.  The location depends on 
where Python and site packages were installed.  Some examples: 

c:\Python36\Lib\site-packages\pypower

$HOME/miniconda3/lib/python3.5/site-packages/PYPOWER-5.0.1-py3.5.egg/pypower
